Dwelt Technical Test Notes

Run the EXE file in this folder to get started!

Welcome to the Dwelt technical test! This build contains no setting or narrative content - it's just a test of the mechanics used to tell the story of the Scrappers and those who dwell amongst the Lattice. Your help in trialing these features, attempting novel combinations of actions, and documenting the bugs that result, will be invaluable in stabilising this world and readying it to be painted in. Checking that saving and loading is robust, combining unforseen actions (for instance, initiating a skill simultaneously from two different inputs), and tests like these will be very helpful.

1.0.2 Release Notes

Running this version will reset your old save and settings. This will only happen during pre-release development.

  • Fixed an issue causing lower FPS values to result in sluggish movement. Slight tweaks to movement dynamics - more speed; reduced slipperiness.
  • Adjusted some visual effects to be more subtle. Various minor fixes.

Thank you so much for your interest in Dwelt! Don't hesitate to contact me directly (slyllama on Discord and Bluesky, or slyllama@outlook.com, etc) with any issues you encounter or ideas you might have. Enjoy the technical test!

Notes For Testers

  • Press F2 to open the debugging menu, which lets you reset your save, hide the player model, and other useful things like that.
  • Settings, save data, and shader caches are stored in your AppData/Roaming/dwelt folder. You can safely delete everything in here, which might be useful if the program gets totally stuck.
  • Please quote the version number always shown in the bottom right of the game when reporting issues and improvements!

And thank you so much for your interest and patience! ♥ —slyllama

Build Dependencies

  • Godot. Dwelt is built in Godot 4.2.2.
  • Jolt 0.12 physics engine is required but not packaged by default. Download it from the Godot Asset Library, place it in the addons folder, and restart the engine. (More information about Jolt on their GitHub page.)

Previous Test Build Release Notes

1.0.1 Release Notes

Running this version will reset your old save and settings. This will only happen during pre-release development.

  • Fixed an issue with the shortcut which prevented the program from opening.
  • Settings are now tabulated, and can be toggled with the ESC key.
  • Added a VSync toggle. Graphics settings are now consolidated into a single "High Quality Graphics" option.
  • Re-entering the updraft while gliding will cause the player to be propelled back upward.
  • Sounds are now perceived from the player model (rather than inaccurately from the camera).
  • Fixed janky behavior when the game loses focus, and other bug fixes.
